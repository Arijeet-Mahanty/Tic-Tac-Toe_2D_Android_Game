# Unity 2D Tic-Tac-Toe
A classic, lightweight recreation of the timeless Tic-Tac-Toe game, built using the Unity Game Engine. This project focuses on clean 2D UI design and smooth gameplay logic, providing a mobile-ready experience.

# 🎮 Game Features
Classic Gameplay: Standard 3x3 grid mechanics for two players.

Responsive UI: Optimized for various screen sizes, ensuring a consistent experience across mobile devices.

Interactive Design: Visual feedback for player turns, wins, and draws.

Lightweight Performance: Built with 2D sprites and optimized C# scripts for high performance.

# 🛠️ Built With
Unity 6000.2.14f1

C#: For game logic and state management.

Unity UI (UGUI): For the menu and game board interface.

# 📲 How to Install (Android)
The game is ready to play on Android devices! You can find the latest build in the Tags (Releases) section of this repository.

Navigate to the Releases/Tags section.

Download the .apk file.

On your Android device, open the file to install (ensure "Install from Unknown Sources" is enabled in settings).

Launch the game and enjoy!

# 🕹️ Controls
Touch/Click: Simply tap or click on an empty cell in the 3x3 grid to place your mark (X or O).

Reset Button: Quickly clear the board to start a new round.

# 📂 Project Structure
Assets/Scripts: Contains the C# logic for game management and board interaction.

Assets/Prefabs: Modular game objects for cells and UI elements.

Assets/Sprites: All 2D visual assets and icons.
