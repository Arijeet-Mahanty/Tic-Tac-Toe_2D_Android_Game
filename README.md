# Tic-Tac-Toe_2D_Android_Game
A simple 2D Tic Tac Toe game built using the Unity Engine. This project was created as a practice project to understand the basics of Unity game development, including game logic, UI interaction, and turn-based gameplay mechanics. Features include 1.Classic Tic Tac Toe gameplay 2.Two-player turn-based system 3.Automatic win detection
